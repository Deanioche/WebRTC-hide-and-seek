# Hide and seek
> Multiplayer game built with WebRTC

## How to run server

```sh
npm install && npm start
```

## Built with

- Node.js
- express.js
- socket.io - signaling Server & hosting game lobby.
- WebRTC - 1:N P2P
- Phaser.JS - rendering client scene.
- Matter.JS - handling physics & collision detection in server.
